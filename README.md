# Portfólio Pessoal - Anisio Gomes

Versão reorganizada em estilo inspirado num perfil GitHub:
- coluna lateral com cartão de perfil
- conteúdo principal à direita
- secções organizadas como um README profissional

## Publicação
1. Criar repositório no GitHub
2. Enviar `index.html`, `style.css` e `script.js`
3. Ativar GitHub Pages em `Settings > Pages`

## Personalizar antes de publicar
- trocar email
- trocar links GitHub e LinkedIn
- substituir avatar pelas tuas imagens
- adicionar links reais para projetos
